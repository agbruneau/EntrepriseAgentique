# Entreprise Agentique

Ce dépôt est une collection organisée de documents relatifs à l'Entreprise Agentique, l'Intelligence Artificielle et l'informatique quantique.

L'Entreprise Agentique est un paradigme où des agents autonomes, orchestrés via une architecture événementielle (EDA) utilisant Kafka, collaborent pour atteindre les objectifs de l'entreprise. Ce concept nécessite une nouvelle discipline appelée "AgentOps" pour gérer le cycle de vie de ces agents. Il existe des liens étroits avec les progrès de l'IA et le potentiel de l'informatique quantique pour résoudre des problèmes d'optimisation complexes au sein de ce paradigme.
